## Karakteristikat e Website-it

### 1. Navigimi (Menu)
- Menu responsive që funksionon në desktop dhe mobile
- Menu mobile që hapet/mbyll me buton hamburger
- Link-et aktive që tregojnë në cilën faqe jeni

### 2. Faqet e Website-it
- **index.html**: Faqja kryesore me slider dhe lajme
- **about.html**: Informacione rreth spitalit, vlerat dhe ekipi
- **services.html**: Lista e detajuar e shërbimeve
- **appointment.html**: Faqja për rezervimin e termineve me doktorë
- **contact.html**: Formular kontakti dhe informacione

### 2.1. Slider i Bannerit
- Slider interaktiv me 4 slide-e në faqen kryesore
- Imazhe në background për çdo slide
- Kalim automatik çdo 5 sekonda
- Navigim manual me butonat dhe dots
- Keyboard navigation (shigjeta majtas/djathtas)
- Touch/Swipe support për mobile

### 3. Funksionalitetet JavaScript
- Menu mobile toggle
- Slider automatik për bannerin
- Animacion i statistikave (counter animation)
- Validim i formularit (kontakti dhe rezervimi)
- Scroll smooth (lëvizje e butë)
- Buton për të shkuar në krye të faqes
- Alert messages (mesazhe informuese)
- Form validation në kohë reale

### 4. Dizajn Responsive
- Funksionon në desktop, tablet dhe telefon
- Grid layouts që adaptohen me madhësinë e ekranit
- Menu që bëhet hamburger në mobile
